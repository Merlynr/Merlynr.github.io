小羊专属男友力报表

发表于2022-07-02|更新于2022-07-02|life
|字数总计:518|阅读时长:1分钟|阅读量:

慎行

  1. 心情不好时，冷静20分钟，再说话做事，记得向小羊说下，不要突然消失
  2. 永远永远不要因为一时不爽就做一些自以为舒服的事，这就是一种不负责任而且还害人害己的事
  3. 与小羊约定好的事就提上日程安排，不要拖拉，不要总是把事情划个三六九等，这样会让事情变质
  4. 每次做事都要先问情自己想要的是什么，你是为了什么，这样在与小羊交流的时候你才不会跑偏
  5. 永远要认清，你是小羊男朋友，以后最依靠的人，这样做事情就要掌握分寸。
  6. 小羊需要的不仅仅是线上虚无缥缈的搞笑，她更需要的是生活中实实际际的关怀和陪伴，是可以感受到的，要记得。

慎言

  1. 少说废话，没用，说话的时候思考一下再说，慢点说，又不花钱
  2. 对爱人要有爱语，不是朋友，摆清楚自个位置，没用的搞笑显得很蠢
  3. 永远不要说一些人格污蔑的话，这是底线，谁都原谅不了你
  4. 有些话少说，可是有些话必须说，不是为了说，是为了陪伴，刷手机看视频的时候记得记录有趣的话题，在小羊上下班的时候，给她看，这样就不孤单了。收藏起来没用的，想做和做是两回事。
  5. 永远不要中断两个人的甜言蜜语，爱人之间，在彼此能获得的东西不多了，千万要珍惜。

  2. 控制体重【TODO 七月要到标准值内】要进行锻炼，长坚实了，硬邦邦的腹肌

4. 完成普通一线大厂工作寻找【TODO 七月八月九月十月】
